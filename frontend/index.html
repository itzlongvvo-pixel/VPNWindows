The Nera Protocol

We don't know who you are. By design.

01. Identity-Less

Others link your email to your payment. We don't. You generate a random 16-digit code locally. That code is your only credential.

02. Toxic Data Gap

Stripe knows your card. Nera knows your code. The two never meet. We use a metadata blind-drop to verify payment without identifying the user.

03. Stateless

No "Login Sessions." No timestamps. Our server performs a simple binary check: Is this code active? Yes/No. No history is retained.
